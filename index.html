¡Bienvenido!

Calcula el material necesario para tus piezas utilizando integrales definidas.

¿Qué son las integrales definidas?

Permiten calcular el área bajo una curva en un intervalo específico. Esta área puede ser interpretada como la cantidad de material necesario para fabricar una pieza con una forma determinada.

Proceso de Cálculo
  1. Seleccionar la forma de la pieza: cilindro, triángulo, aro...
  2. Definir la función: matemática que describe el área.
  3. Establecer límites: inferior y superior.
  4. Calcular el área: usando la integral definida.
  5. Interpretar el resultado: para conocer el material requerido.
